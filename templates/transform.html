Semantic Concepts

Extract general concepts from word pair examples, then apply them to new words. For example, if you provide pairs like cow→beef and pig→pork, the system learns the "animal to food" concept by averaging the vector differences. You can then apply this concept to deer to get venison. Try it with countries→capitals, verbs→past tense, or any pattern you can think of.

Step 1: Enter Word Pairs

Enter pairs in format: word_from,word_to (one pair per line). The system will learn the transformation pattern from these examples.
